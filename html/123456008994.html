Product / Service / Licensing

Product Name:
S-FRAME for Windows
2D or 3D Structural Analysis Software. Unique 2D/3D switch: P-
FRAME Analysis package is built into 3D S-FRAME. Supports
membranes, quadrilateral/triangular plates, shells, non-
sequential joint numbering. Automatic generation of trussess,
multi-storey frames, arches, circles and spirals. Translate,
Rotate, Scale and Reflect Editing Features, Graphical Section
Editor, Two-way DXF Interchange, Undo/Redo, Moving Loads, P-
delta, Non-linear Newton Raphson Iterative Solver with
Incremental Loading (Tension/compression only, cable, hook and
gap elements), Buckling, Stressed/Unstressed Vibration, Seismic
Analysis, Dynamic Analysis, Segmental Force Envelopes, Influence
Lines. Wind load generator in accordance with API Specification 4F:2008

Product Name:
S-STEEL for Windows
3D Steel Design Program to complement S-FRAME or P-FRAME for
Windows. Fully integrated, iterative design and analysis
process. Code Checks or automatic redesign options for all or
select part of structure. Member grouping for rationalised
designs. Scratch pad for design refinement. Lists of Admissible
Sections, Quantily Takeoff, Structure Costing. Supports American
ASD-89, LRFD-94, Canadian S16.1-M94, New Zealand NZ 3404,
Australian AS 4100, European EC3, and British BS 5950 Codes.
Generates Reports on Material Takeoff, Total Weight, Cost,
Surface Area. Full representation of Results.
