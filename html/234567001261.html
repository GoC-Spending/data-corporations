Company Profile


Neatware is an Independent Software Vendor (ISV) and a Solution Provider (SP) located in Toronto, Canada. Its mission is to innovate and develop neat software and create values for users. Neatware provides software products and solutions in mobile application, video processing, data visualization, and enterprise application. Neatware has worldwide users from Network, Telecom, Resource, and Financial sectors to normal consumers.

Product / Service / Licensing

Product Name:
Snaml Book
Snaml Book is a 2-Dimensional Web Organizer, a 4-Way Web Navigator, and a full screen Web Browser for iPhone, iPad, and Android Phone/Tablet. Snaml Book organizes web sites in the architecture of Library, Category, and Page. Each Page can be a HTML5 page. A Category includes a group of pages. And a set of Snaml Book files are stored in a database as a Library.

Snaml Book allows you to organize web sites in category, browse and preview them by swiping in four directions on iPhone. You can manage and navigate HTML5 pages, PDF documents, JPEG photos and H.264 movies easily through Snaml Book.

KEY FEATURES

Support iOS 5 features like iCloud and Twitter
Organize Web Sites in a Dynamic Book
Easily Add, Delete, Exchange Cards
Convenient Navigation by Swipe Preview
Snaml Cross for Quick Navigation Display
Support HTML5 and Video Playback
Full screen Web Browser and Media Navigation
Organize Photo and Video from Camera Roll
Library for Snaml Book View, Save, and Share
Share Snaml Book by Email, Twitter, Web


Product Name:
Ladybug Mixer
Compositing live and vivid videos, showing stunning visual effects in high-definition, Ladybug Mixer, a multi-channel media mixer with real-time visual effects on Windows, works greatly for your video processing and application such as video remix, digital signage, video server, and streaming video.



Product Name:
MyrmecoX Studo
MyrmecoX Studio is a suite for Tcl development. The latest MyrmecoX Studio 4.0 includes an IDE (tcl 8.4, editor, checker, debugger, keyword editor, and byte code compiler ) and integrated extensions for interactive automation, object-oriented programming, java integration, database connection, graphs and charts, COM interface, exception handling, extension wizard, as well as advanced gui (expect, itcl, blt, tcom, trycatch, xwizard, bwidget). Finally, MyrmecoX Studio supports SnamlScript and Angular 2 on Web Application Development.

Product Name:
Ladybug Player
Ladybug Player is a popular 4K HEVC, H.265, H.264, AVI, VCD, MPEG, video and MP3, WAV, MIDI audio player. It supports video playback in full screen and full speed. Ladybug Player can change the rate of playback. Furthermore Ladybug Player supports multi-tasking, multi-screen, and hardware video acceleration.

Product Name:
Snaml Theater
Snaml Theater is a Cloud TV app for iPhone, iPad and Apple TV. It is also a Windows 10 Universal Application. A new generation TV architecture is designed for Cloud video streaming and using. It completely change the use of TV and Streaming video. By using Snaml Theater you can ultra simply change thousands channels by swipe the remote control of Apple TV. All programs and videos are managed on cloud database from smart phone and tablet.


Service Name:
Neatware IT Services

testing, system tuning and optimizing, as well as system security configuration.

Technology profile

1. Software

Neatware's software products work for Entertainment, Enterprise, and Internet.

Entertainment
Ladybug Mixer is a software family for multimedia applications.
Ladybug Player is a fast, simple, and reliable video player.

Enterprise
MyrmecoX is a rapid development environment for enterprise applications.

MyrmecoX Professional and Enterprise worked for application integration, data minning, administration automation, and network / chip / device testing. MyrmecoX IDE includes an engine, editor, checker, debugger, and compiler. MyrmecoX Runtimes work for cross-platforms. Developers can use MyrmecoX to complete application integration with the help of Java integration, database connection, and GUI package. Furthermore, users can rapidly develop their own testing programs for embedded system, devices, and network by using MyrmecoX.

Internet
SnamlScript is an Open Source HTML5 generator that supports HTML5, Javascript, and Angular 2 framework. It also support In-Memory Web Server to dramatically speed up web application.

Snaml for Java components work for J2EE platform. Developers can use Snaml to build and manage the contents on web and
mobile site. Snaml supports open standards (such as XML, XHTML,
and WML).

2. Solution

Neatware provides solutions for enterprises and individuals to meet their demands on application integration, storage, wireless, and entertainments. Neatware integrated and optimized its software along with its partners' products to bring customers the latest technology and great values.

Entertainment Solution

Neatware provides entertainment solution packages with its Ladybug Mixer and partners' products. These integrated
packages bring customers digital movie experience by integrating Ladybug Mixer and latest Microsoft DirectX, fast Intel Core processor as well as GPU.

Enterprise Solution

Neatware provides enterprise solutions for corporations. Network, chip, instrument, and embedded system developers can
complete their automation testing and GUI building with MyrmecoX. Biotech and Financial users can use MyrmecoX to do data mining on big data with 64-bit processors.

Internet Solution

Neatware provides Internet solution for users with its SnamlScript software for Web applications. SnamlScript can help enterprises to build fast web and mobile application. Neatware helps customers to use latest technology to reduce cost and increase the ROI for their business on Internet.
