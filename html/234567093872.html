Company Profile


WNL provides complete identity management and network security with a single protocol Dynamic Identity Verification and Authentication and one scalable and interoperable software framework - Dynamic Distributed Key Infrastructures.

One-time-pads are the only provable, unbreakable key technology.

One key creates an infinite number of one time pads. DIVA is differentiated by comparing tokens ahead in a key stream that have never been created or used before and is continuous throughout a network session. It is secure, simple, inexpensive, virtually manufactured and provisioned, with a scalable framework, scalable exponential keys>250,000 bits, key streams > 10 to the 60th power, nothing to replace and integrates into any sign-on routine or application with a single call.

One key provides all network security controls including: perfect identity, secure network access, continuous dynamic authentication, authorization, signature, non-repudiation, inherent intrusion detection and automatic revocation.

Whitenoise Technologies fix all asymmetric network security flaws. All the following attack classes are prevented:

* Man-in-the-Middle attacks are prevented because there is no key exchange

* Side Channel attacks are prevented because all operations are order 1 after key load and because there is no access to the key

* Botnet attacks are prevented by configuration with server so the botnet never has access to the entire key and offset information.

* Quantum computing attacks are impossible because every variable is variable.

* Brute force attacks are prevented because the keys cant be factored.

* Denial of service attacks can be prevented by exploiting unbreakable identity and secure network access so that hackers could never get on a network.


We are patented globally in countries with two thirds of the worlds population, revenue and IT manufacturing. Over 40 patents have been granted in the US, EU, Canada, China, India, etc. Many more are in process. The patents cover Whitenoise, DIVA, DDKI, preventing Man-in-the-Middle, securing the cloud, mitigating false positives and negatives in biometrics, preventing internal and external theft etc. The patents create a significant IP base and allow for the licensing of technologies to geographic areas.

Whitenoise Laboratories (Canada) Inc. have introduced a new framework for cyber security that uses two protocols that they have created - Dynamic Distributed Key Infrastructures (DDKI) and Dynamic Identity Verification and Authentication (DIVA). Taken together, these two approaches serve to harmonize identity management, privacy and security issues.

DIVA prevents Identity and Intellectual Property theft by providing true identification and authentication throughout a network session. The protocol has inherent intrusion detection capabilities and automatically denies unauthorized network access.

Product / Service / Licensing

Product Name:
Secure File Interchange 2
SFI provides the enterprise with an extremely secure, highly functional, document exchange system at very affordable pricing. Its intuitive graphical user interface, when combined with the highly secure Identity Management features of Whitenoise, provides an integrated document security system that people will use because it doesn't slow them down.

SFI provides multi-layered user access security. Users are issued a unique USB access key that employs user ID and password protection as well as the requirement for the physical presence of the USB Key. The key allows access to the system both locally and remotely. Keys can be issued to partner companies as well. The keys are used to encrypt sensitive information on the individual's computer or computers. This provides additional security in the event a user's computer is lost or stolen.
Secured information is exchanged as per predefined rules. Information can be sent to one or more recipients, or predefined groups, in one operation.

SFI is simple to implement and is designed to fit within existing security schemes. It is under the control of the system administrator. A very important feature of the system design is the ability for the system administrator to deactivate lost or stolen keys immediately.
If you would like more information on SFI please contact us at sales@whitenoiselabs.com and give us your contact information to have someone contact you or to receive a copy of our CD-ROM Overview and Demonstration of SFI.

It can be used with AES and Dynamic Identity Verification and Authentication identity management keys.
