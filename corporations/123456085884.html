Company Profile


Spectrum Signal Processing designs and builds board and system-level hardware solutions for demanding signal and video processing applications. Spectrum combines high-performance data acquisition (RF, analog and digital I/O) and reconfigurable signal and video processing hardware with its best-in-class support and engineering services to enable customers to rapidly develop and then deploy their products and systems. Spectrum has worked with both commercial and military customers worldwide to develop and deliver solutions optimized for Intelligence, Surveillance and Reconnaissance (ISR), video, military (MILCOM) communications and satellite communications (SATCOM) markets. For more information on Spectrum and its products, please visit http://www.spectrumsignal.com. Spectrum is part of Vecima Networks Inc and its products are designed and manufactured in Vancouver, British Columbia, Canada. See http://www.vecima.com for more information on Vecima.

HCDR-1000 CRDP for Communications Intelligence (COMINT)

Product Name:
Public Safety Cross Band Communications Platform
Spectrums Public Safety Cross Band Communications Platform uses SDR technology to enable interoperable communications across multiple radio frequencies allowing first responder services to share critical information. The SDR architecture of the solution supports advanced communications capabilities such as MIMO and cognitive radio that enables efficient use of available frequency spectrum. The wireless communications platform is compact and can be ruggedized for operation in many types of harsh environments, allowing rapid deployment in a disaster area when primary communication systems are rendered inoperable.

The platform is a commercial-off-the shelf solution integrating a high performance IF and baseband processing subsystem with two independent RF subsystems: one operating in the Ultra High Frequency (UHF) band between 350-800 MHz, and one operating in the 4.9 GHz public safety band.

For more information, please visit http://www.spectrumsignal.com/products/sdr/publicsafety_crossband_platform.asp.

Product Name:
UHF SATCOM MIL-STD-188-181B waveform implementation on the SDR-4000
Ultra High Frequency Satellite Communications (UHF SATCOM) provides military users with a long-haul data and voice communications capability. To reduce your technology risk and speed your time to market, Spectrum offers the physical layer (PHY) implementation of the UHF SATCOM MIL-STD-188-181B waveform on its flexComm SDR-4000 rugged platform.

The modulator/demodulator offering data-modes described in MIL-STD-188-181B is implemented as a software application on the SDR-4000. A convenient 5 KHz or 25 KHz channelization carries the spectrally efficient digitally modulated information waveform. A suite of multi-h continuous phase modulation (CPM) modulated data-modes complements a variety of linearly modulated modes (PSK, OQPSK, and FSK), offering the user and the application a chance to trade-off power (or alternatively range) and data rate. In the receiver, robust waveform acquisition signal processing establishes timing and phase synchronicity, and maximum likelihood-based estimation techniques are used to ascertain and correct carrier-slips during ongoing phase tracking. CPM data is demodulated using Viterbi techniques, where the memory inherent in the CPM modulation allows excellent detection performance. For added robustness, optional data modes including those with forward error correction (FEC) and interleaving are offered.

For more information, please visit http://www.spectrumsignal.com/products/waveforms/uhf_satcom.asp

Product Name:
Sensor Data Link Modem System
Sensor-to-ship data link communications for protection and monitoring of littoral waters
==================================

Spectrums Sensor Data Link Modem System (SDLMS) provides a deployable IF-to-Ethernet modem solution for the radio data link between a shipborne transceiver and remote sensors deployed in littoral waters, such as in passive, bottom-fixed, battery-powered hydrophones, unmanned underwater vehicles (UUV), unmanned surface vehicles (USV), and unmanned airborne vehicles (UAV).

The SDLMS consists of two sub-systems: a software defined radio modem transceiver subsystem for the shipborne radio gateway, and one or more remote sensor modem transceiver subsystems for the sensor radios. The default configuration for the shipborne transceiver consists of 8 channels of IF input and output utilizing software reconfigurable processor technology such as field-programmable gate arrays (FPGA), digital signal processors (DSP) and general purpose processors (GPP) to run data link waveforms such as STANAG 7085, Tactical Common Data Link (T-CDL), Wideband Networking Waveform (WNW) and 802.16 (WiMAX).

The remote transceiver subsystem of the SDLMS is packaged in a small 3U CompactPCI form-factor that is conduction-cooled and ready for integration into your remote sensor package. The remote transceiver subsystem features a processor architecture that is virtually identical to the base transceiver architecture consisting of software reconfigurable processor technology combined into an integrated package that is size, weight and power optimized for deployment in harsh environments, and where power is limited.

For more information, please visit http://www.spectrumsignal.com/products/platforms/sensor_data_link_modem_system.asp.

Product Name:
SDR-4000 TMRDP Tactical MILCOM Rapid Deployment Platform

The SDR-4000 Tactical MILCOM Rapid Deployment Platform (TMRDP) is a fully integrated, black-side RF-to-Ethernet software defined radio (SDR). This integrated COTS platform is designed specifically to de-risk customers programs by providing quick response capability for deploying Software Communications Architecture (SCA)-enabled radios, waveform applications, and verification and validation platforms.

The SDR-4000 TMRDP offers the full modem processing functionality of Spectrums SDR-4000 IF-to-Ethernet wireless modem combined with a DRTi high performance fast tuning RF transceiver. The transceiver and modem are co-located in a 19-inch rack. In addition, the SDR-4000 TMRDP supports the Communication Research Centre Canada (CRC) SCARI++ implementation of the SCA v2.2. System software examples with source code provide a starting point for developing receive and transmit path functionality. Processor-to-processor data flow examples with source code are provided to further reduce your time and cost to deployment.

For more information, visit http://www.spectrumsignal.com/products/platforms/sdr_4000_tmrdp.asp.

Product Name:
RF-4902 High Linearity Wideband RF-to-Digital Transceiver
The RF-4902 is a wideband, frequency-agile RF-to-Digital transceiver designed to provide customers with cost-effective, high-performance RF capability in a compact 3U form factor. It features a high linearity, wideband RF tuner/exciter ranging from 200 MHz to 2.7 GHz (UHF, VHF, L-Band, ISM) for use in a wide range of applications including wideband datalinks, tactical communications, signals intelligence (SIGINT  COMINT/ELINT), and software defined radio (SDR).

Product Name:
XMC-1151 Wideband Digital Receiver/Digitizer Module
The XMC-1151 is an ultra high-speed digitizer and processing solution in a VITA 42.3 XMC form factor. The module allows for direct RF-to-digital baseband conversion using a single dual-channel 12-bit ADC running at 1.6 GSPS or 3.2 GSPS when both channels operate together in interleaved mode. The ADC conversion stage is integrated with a Xilinx Virtex-6 LX240T FPGA device and 1 GB of DDR3 SDRAM for wide bandwidth signal processing and filtering in the digital domain. The module has an 8-lane PCI Express, ANSI/VITA 42.3 host interface providing up to 4 GB/s full duplex to the host.

Technology profile

Spectrum provides advanced technology based on the latest processor architectures and software standards for the purpose of processing wireless signals, whether they are commercial air interface protocols like CDMA or military waveforms like the Wideband Networking Waveform.

Fundamentally, Spectrums products are built around the principle of choice providing optimal processing platform configuration for customers' specific development and deployment requirements. Customers can choose the type of processing element selected for each stage of their algorithm, the number of processors per stage, the configuration and types of dataflows between processors and stages, the level of hardware abstraction or programmability, and even the software development tools.

Spectrum was one of the first companies to recognize the changing paradigm of high performance to embedded computing platforms from homogeneous processing (i.e., the use of a single type of processor) to heterogeneous processing (i.e., the use of multiple types of processors). Allowing different combinations of general purpose processors (GPPs), Digital Signal Processors (DSPs), and Field Programmable Gate Arrays (FPGAs) provides solutions based on customers requirements for performance, programmability, power consumption and cost.

Spectrum also recognized the movement from dedicated resources, where an entire processor or processors are dedicated to a specific algorithm or waveform, to shared resources, where processors in the system may be supporting multiple algorithms or waveforms at the same time, resulting in a more cost effective and efficient system. Spectrum can support both models with its technology, an advantage that will pay off in 2004 and beyond as more communications systems move to a shared resources model.

Spectrum has created a layered software operating environment that allows customers to program at their desired level of hardware abstraction. This environment also encompasses the Software Communications Architecture (SCA) mandated for all military communications programs going forward. As a result, Spectrum is a recognized industry leader in the development of processing engines supporting software defined radios.

· Software Communications Architecture (SCA) acceptance and compliance testing of JTRS waveforms
· Supporting the advancement of the Software Communications Architecture
· Providing growth capabilities for the acceptance and compliance testing of future JTRS waveforms.

Technology

Technology profile

Spectrum provides advanced technology based on the latest processor architectures and software standards for the purpose of processing wireless signals, whether they are commercial air interface protocols like CDMA or military waveforms like the Wideband Networking Waveform.

Fundamentally, Spectrums products are built around the principle of choice providing optimal processing platform configuration for customers' specific development and deployment requirements. Customers can choose the type of processing element selected for each stage of their algorithm, the number of processors per stage, the configuration and types of dataflows between processors and stages, the level of hardware abstraction or programmability, and even the software development tools.

Spectrum was one of the first companies to recognize the changing paradigm of high performance to embedded computing platforms from homogeneous processing (i.e., the use of a single type of processor) to heterogeneous processing (i.e., the use of multiple types of processors). Allowing different combinations of general purpose processors (GPPs), Digital Signal Processors (DSPs), and Field Programmable Gate Arrays (FPGAs) provides solutions based on customers requirements for performance, programmability, power consumption and cost.

Spectrum also recognized the movement from dedicated resources, where an entire processor or processors are dedicated to a specific algorithm or waveform, to shared resources, where processors in the system may be supporting multiple algorithms or waveforms at the same time, resulting in a more cost effective and efficient system. Spectrum can support both models with its technology, an advantage that will pay off in 2004 and beyond as more communications systems move to a shared resources model.

Spectrum has created a layered software operating environment that allows customers to program at their desired level of hardware abstraction. This environment also encompasses the Software Communications Architecture (SCA) mandated for all military communications programs going forward. As a result, Spectrum is a recognized industry leader in the development of processing engines supporting software defined radios.

Unique applications:

The U.S. Department of Defense Joint Tactical Radio System (JTRS) Joint Program Office (JPO) selected Spectrum's <i>flex</i>Comm SDR-3000 platform as the only Commercial-Off-The-Shelf (COTS) representative JTR hardware set. The JTRS JPOs test and certification element, the JTRS Technology Laboratory (JTeL) will be used for independent verification and validation of JTRS waveforms for the following purposes:
· Software Communications Architecture (SCA) acceptance and compliance testing of JTRS waveforms
· Supporting the advancement of the Software Communications Architecture
· Providing growth capabilities for the acceptance and compliance testing of future JTRS waveforms.
