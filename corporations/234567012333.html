Company Profile


PresiNETs Enterprise Performance Management Solutions provide a
real-time window into the interrelationship of the many
applications, complex technologies, security events, and users
deployed on the network utilizing the Internet, producing a
unified, enterprise-wide picture. This allows you to manage your
network as a whole entity, and allows for detailed analysis at
the corporate, office and workstation levels.

PresiNETs Enterprise Performance Manager seamlessly unites the
core performance management and security disciplines 
Application Monitoring, Internet Gateway Monitoring, Connection
Tracking, Security Event Management, Capacity Planning,
Troubleshooting and Fault Prevention  consolidating the multiple
performance and security reporting/monitoring/management tools in
your network and Internet gateway; provided to you as a
co-managed service or enterprise product.

Technology profile

Application Monitoring
Comprehensive Identification of Application Traffic

PresiNET Enterprise Performance Manager provides the most
complete visibility into how applications are being used to and
from the Internet and how they impact total network performance.
This application-level perspective lets you make real-time
quantifiable business decisions about your network resources and
bandwidth.

Internet Gateway Monitoring
Enterprise-Wide Network Visibility

PresiNETs Enterprise Performance Manager provides comprehensive
information across all the Internet gateways, giving you optimal
visibility enterprise-wide, regardless of number of locations,
time frame, geography, or underlying network technology.

Connection Tracking
Detailed Audit and Policy Management features

PresiNETs Enterprise Performance Manager provides detailed
Connection Tracking information to help Human Resource Managers
develop and enforce appropriate Internet Usage Policies. For
Network Managers, detail Audits are available to support
Bandwidth Consumption and aid in tracking down unidentified
application activities.


Security Event Management
Enterprise-Wide Security Event Visibility

PresiNETs Enterprise Performance Manager displays Security
Critical Events in Real-Time, enabling quick and accurate
responses by PresiNET to help in managing the Enterprise Security
Requirements. Detailed Critical Event information is displayed
to allow an Enterprise-Wide overview of security threats, a
detailed analysis of Internet attackers, and information to help
track down the source and intent of attackers.


Capacity Planning
Plan for Growth and Manage your Budget

PresiNETs Enterprise Performance Manager offers detailed
forecasts for calculating Growth Trends and identifying when and
where bandwidth will run out, helping you to avoid costly network
congestion. PresiNET goes beyond basic utilization data to show
you which applications consume bandwidth, providing the
justification for your capacity planning decisions and allowing
you to keep your costs in check.


Troubleshooting
Fast and Effective Service restoration

When network degradations and outages occur, PresiNETs
Enterprise Performance Manager helps you rapidly isolate and
resolve problems so that you can restore services as quickly as
possible, minimizing the effect on your organization. Task-Flow
Problem Resolution lets you track the precise metrics,
applications, locations, and time frames you need to solve your
problems.


Fault Prevention
Alarm on Performance Issues before users notice

PresiNETs Enterprise Performance Manager can proactively
identify where Internet and application problems are starting to
occur so you can prevent performance degradations from reaching
critical levels.


Co-Managed Solution
Shared Support for Internet and Security Management

PresiNETs Enterprise Performance Manager provides the ability to
share the Internet and Security Management and Configuration with
PresiNETs Secure Operations Centre. Your organizations
Information Technology staff can perform remote configuration
changes from within the organization, or from home. You can also
choose to have PresiNET perform these functions through our
Security and Network Experts on your behalf.


Firewall Security Solution
Integrated Internet Security Management

Based on our vDeadbolt Enterprise Appliances, the Managed
Firewall technology protects any TCP/IP based network. Our Secure
Operations Center monitors all firewall activity, responds to
critical events, maintains and updates the firewall systems, and
provides detailed activity reports 24x7x365. The managed VPN
technology provide affordable and secure communications "tunnels"
between multiple sites and remote users, regardless of their
geographical location on the Internet. Other technologies
included in the solution are Intrusion Detection, redundant
failover, high availability and more. The functions of the
firewall Security Solutions are tied into the Enterprise
Performance Manager to create a truly unique security and
management offering.
