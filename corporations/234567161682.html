Company Profile


Arcturus Networks Inc. is a leading provider of embedded enablement solutions for communication and industrial applications. Recognized as a pioneer, Arcturus has contributed to the innovation of thousands of products and has helped establish embedded Linux as a defacto global standard.

Since 2002, Arcturus has focused on middleware solutions to enable device management, M2M, IP audio and IP voice communication, including audio intelligibility and psychoacoustics. Arcturus middleware, reference designs and modular hardware systems reduce time-to-market and overall project risk by combining proven solutions with direct support.

Arcturus has numerous industry affiliations and partnerships, mostly notably with top semiconductors; Freescale, Analog Devices, Conexant and the ARM connected community. Arcturus works closely with its semiconductor partners to deliver specialized hardware with bundled middleware packages – in some cases licensing middleware as part of the semiconductor ASP.

As a vertically focused company Arcturus strives to be on leading edge of innovation in healthcare, transportation and buildings. System Solution packages tailored toward these key markets focus on devices as part of an overall system (Internet of Things), supporting work flow, mobility, supervision and maintenance. Arcturus System Solutions are used today to enable IoT applications in connected home, healthcare and public transportation environments.

Product / Service / Licensing

Product Name:
uCMK64-IoT Microcontroller System on Module
The uCMK64-IoT is a 60x60mm module for developing secure IoT devices that require a combination of connectivity and control. The hardware uses a 120MHz, NXP Kinetis K64 microcontroller with Ethernet, Wi-Fi, TLS security, peripheral connectivity and optional audio. The platform is controlled using a simple command protocol over a UART or TCP/IP socket, providing options for both host-MCU or cloud integration. The protocol supports I/O, bi-directional UART-to-net communication, device services and settings. A “call home” feature automatically originates the secure TLS socket connection to a remote server, helping to egress firewalls.

Product Name:
uCP1020 Industrial IoT Gateway and Communications System on Module
The uCP1020 is a compact (80mm x 80mm) System-on-Module (SoM) designed for communication and data delivery applications that require a combination of CPU performance, high-throughput network processing and rugged operation. The module uses an 800MHz dual-core NXPTM P1020 PowerTM architecture communications processor. The system features up to 2x Gbit Ethernet networks and 2x PCIe lanes for additional Wi-Fi, 3G/4G or other data communications. The 120-pin uCP1020 module supports all necessary sub-systems on-board, it uses industrial rated parts and requires only passive cooling, no spinning disc and no fan.

Product Name:
uCMK60 Microcontroller System on Module
The uCMK60-VoIP is a microcontroller (MCU) based solution for VoIP and IP audio distribution. The hardware uses a 120MHz NXP Kinetis K60 microcontroller (MCU) and supports narrow or wide-band voice communication using VoIP (SIP) or multicast paging. The system includes specialized audio intelligibility and psychoacoustic functions necessary to support high-fidelity, duplex intercom and speakerphone applications. VoIP and Mbarx IoT middleware is preloaded and executes internal to the MCU, with no requirement for external memories. The platform is operated using dedicated I/O controls or can be integrated into an overall system using the Mbarx ASD protocol on a dedicated UART or socket connection. An ecosystem of tools is provided including Mbarx System Manager for site-wide configuration and maintenance.

Product Name:
uC5328 Microprocessor System on Module
The uC5328 is designed for general purpose, machine interface, sensor, control and protocol conversion applications. The module is a highly integrated solution with a mix of HMI LCD, I/O and peripheral device connectivity making it suitable for the development of building control, point-of-sale, networked sensor or industrial control devices. The uC5328 is a second generation embedded platform maintaining pin compatibility with first generation uC68VZ328 modules.

Product Name:
uC53281-EVM Embedded Voice Module
The uC53281EVM module is a hardware platform designed to help developers enable high-quality SIP based VoIP products without having to become media, audio and signaling experts. The software package for the module (available with the kit) features a complete VoIP and Management Middleware system and an open source uClinux BSP. The module integrates into end product by using a simple 50 pin connector and includes all necessary subsystems to support full duplex audio, network, I/O and serial connections with minimal external circuitry.

Product Name:
uCBF548-EMM Embedded Media Module
The uCBF548 embedded media module is a highly integrated system-level solution for the creation of a wide range of mixed two-way voice, broadcast audio, mini-PBX or video applications. The module combines the value of low-cost per channel IP audio with the scalability to support video, making it an ideal platform for digital IP media devices. It features complete Voice and Management Middleware bundled with an open source uClinux BSP to enable high quality voice support without the necessity of having to become a media, audio or signaling expert. The device integrates into end product by using a simple 200 pin soDIMM edge connector and includes all necessary subsystems, requiring only minimal external circuitry.

Product Name:
uC5275 Microprocessor System on Module
The uC5275 embedded microprocessor module is a second-generation device that enables product developers to create advanced networking and control equipment including secure data, control and protocol conversion devices. It features an advanced 150MHz NXP ColdFire processor with dual network controllers capable of supporting fibre or copper network infrastructure. The platform supports multiple commonly used serial protocols as well as various I/O and peripheral connectivity options for the connection of specialized devices. The module integrates into end product using a snap-in soDIMM form-factor connector and is suitable for use in harsh environments. The software includes uCbootloader, and a uClinux BSP.

Product Name:
IOCOM Intercom and Audio Paging Platform
The IOCOM is an installation-ready, board-level platform for IP voice and distributed audio communication including VoIP, intercoms, PA paging and mass notification. The system includes Mbarx-M2M to support applications that require deep supervision or remote operational control by a central host.

The device accepts PoE or 12VDC supply and supports amplified audio output of up to 5W (using PoE) and up to 15W using supply voltage. The board has isolated contacts and loop closures for connectivity to push-buttons, indicators, door controllers or other equipment.

The hardware supports an audio subsystem capable of supporting intelligibility and psychoacoustic enhancements including Acoustic Echo Canceller (AEC), Auto Gain Control (AGC), noise reduction and ambient noise compensation.

The IOCOM is available as part of a System Solutions engagement packaged with development hardware, direct support, software customization and Mbarx reference code for host integration. The platform is compatible with the Mbarx tools suite including the System Manager maintenance utility and Operations Controller for developing integrated work flow process.

System Solutions engagements help accelerate the creation of specialized voice communications products in buildings, transportation or healthcare environments, including access control, rescue assistance, nurse call and mass notification.

Product Name:
Mbarx-M2M Embedded Endpoint Software and PC Host Tools
Mbarx-M2M is embedded end-point software with a suite of PC host tools for machine-to-machine (M2M) communications. It is suitable for site-wide operation and maintenance of many networked devices with various capabilities, using a private cloud model. Mbarx is suitable for applications including:

Building access and security
Healthcare and patient care systems
Mass transit system control and security
Data acquisition and SCADA devices
Connected sensors and meters
Building automation, HVAC systems and controls

Product Name:
uC5282 Microprocessor System on Module
The uC5282 embedded microprocessor module enables product developers to create multiprotocol control and conversion devices including CAN products suitable for harsh EMI environments. It features a NXP ColdFire processor with Ethernet support as well as various I/O, serial and peripheral connectivity options. The module integrates into end product using a snap-in soDIMM form-factor connector and is suitable for applications with extreme temperature ranges. Software includes uCbootloader, a uClinux BSP.

Product Name:
uC5272 Microprocessor System on Module
The uC5272 embedded microprocessor module enables product developers to create network routing, monitoring or control equipment including secure data devices. it features a NXP ColdFire processor with dual on-board Ethernet support and various I/O and peripheral connectivity options. The module integrates into end product using a snap-in soDIMM form-factor connector and is suitable for harsh environments. Software includes uCbootloader, a uClinux BSP.
